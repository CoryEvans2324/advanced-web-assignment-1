Web Performance & Maintenance

Performance

  • Reduce the size of your images to match the dimensions they are displayed at.
  • Enable browser caching of your static content.
  • Minify CSS, JavaScript and HTML content.

Maintenance

  • Test your website and it's functionality regularly.
  • Make sure information is correct and up to date.
  • Make regular backups of your website.

Web security

5 common cyber threats

  • DOS & DDOS attacks

    Denial of Service attacks can be prevented by using a service like Cloudflare

  • MITM Attacks

    Man In The Middle attacks can be prevented by enforcing https and setting up SSL certificates. letsencrypt provide free SSL certificates.

  • XSS

    Cross Site Scripting can be mitigated by sanitising all user input.

  • SQL Injection

    Sanitising user input and using a seperate ORM library to manage the contrustion of SQL statements.

  • General & Zero Day Exploits

    The best and easiest thing anyone can do to stay safe online is to install the latest updates for the software you are using.
